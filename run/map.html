Running Challenges Map

This is a map of various running challenges in the UK. I am focusing (to start with) on the North of England, because that's where I'm based. When I'm looking for inspiration for big fun days out, it's tricky to see what's already been done, and where these routes are. Hopefully this is a more useful compedium. The aim is to show routes that are neither race routes nor long distance paths marked on OS maps. Much of these are sourced from the Go Far website, the LDWA, as well as the FRA forums, and various running groups on Facebook.
Key: <30 miles; 30 - 50 miles; >50 miles
